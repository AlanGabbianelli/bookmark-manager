# BOOKMARK MANAGER

```
As an internet user,
So that I can remember my bookmarks,
I'd like to see a list of links on a website.

As an internet user,
So that I can remember more bookmarks,
I'd like to add more links to the website.

As an internet user,
So that I can organize my bookmarks,
I'd like to be able to add tags to my bookmarks.

As an internet user,
So that I can find my bookmarks easily,
I'd like to be able to filter them by tags.
```
